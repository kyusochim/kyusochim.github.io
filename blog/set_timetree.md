# TimeTreeの更新を少しでも楽にしたい
## 経緯
TimeTreeといえば、共有できるカレンダーとして
所属グループの予定管理に重宝している方もいるのではないでしょうか。
でも複数カレンダーに同じ予定を登録するの…面倒じゃありません？
カレンダーAの内容は全部カレンダーBに共有したいとかあるじゃん。
でも他のカレンダーに登録したやつを一気には別カレンダーに転記できないんですよね。仕様で。
ちょっとでも楽にならないかの試行錯誤記録です。


## iPhone標準カレンダーを使用して複数予定の追加を一度にする
### TimeTreeの仕様
詳細は[公式](https://support.timetreeapp.com/hc/ja/articles/360000639742-%E4%BB%96%E3%81%AE%E3%82%AB%E3%83%AC%E3%83%B3%E3%83%80%E3%83%BC%E3%82%92%E5%85%B1%E6%9C%89%E3%81%97%E3%81%9F%E3%81%84-Google-%E3%82%AB%E3%83%AC%E3%83%B3%E3%83%80%E3%83%BC%E3%81%AA%E3%81%A9)を見て欲しいんですが…
TimeTreeはOS側の標準カレンダーからならインポートできるらしいんですよね。
エクスポートはできないけど。

### iPhone標準カレンダーについて
iPhone標準カレンダーからのインポートはできるので、複数予定を一気に入れたい場合はこれを利用した方が楽です。

ちなみにiPhoneの標準カレンダー、今までろくに使ったことなかったんですが…
複数カレンダーに分けられるんですよね。

### TimeTreeへのインポート
例えば以下のように標準カレンダーで分けて使用できます。
1. プライベートの予定
2. 学校/仕事の予定

これを家族には1,2両方共有
学校/仕事のつながりの人には2のみ共有っていうやりかたができます。

共有したい先のTimeTreeのカレンダー
→右上の設定(四角4個)
→外部カレンダーの予定をインポート
→インポート元のカレンダーを選択(TimeTree側の予定の色も指定)

で簡単にできます。
